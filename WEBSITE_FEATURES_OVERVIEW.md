# AI Intelligence Network - Website Features Overview

## Main Dashboard (Command Center)
- **Location**: `/` (Homepage)
- **Features**:
  - System status monitoring with real-time metrics
  - Intelligence coverage statistics
  - Data quality indicators
  - Universal language translation button
  - Quantum CAI Chat integration

## News Feed System
- **Location**: `/news`
- **Core Features**:
  - Live AI, Tech, and Finance news aggregation
  - Multi-country news sources (30+ countries)
  - Real-time article filtering and categorization
  - Professional horizontal button layout (no carousel)

### Action Buttons (Horizontal Layout)
- **SUBMIT AI ARTICLE** (Gold theme) - Links to AI Fans Race
- **AI NEWS** (Pink theme) - Right-side popup with startup news
- **FANS AI** (Green theme) - Bottom popup with community articles
- **JOB SEARCH** (Blue theme) - Left-side popup with job listings
- **Command Center** (Red theme) - Returns to main dashboard

### Popup Systems
- **AI NEWS Popup**: Slides from right edge of screen with pink beam animation
- **JOB SEARCH Popup**: Slides from left edge of screen with blue beam animation
- **FANS AI Popup**: Slides from bottom with green hacker animation
- All popups are positioned outside the main content area for immersive experience
